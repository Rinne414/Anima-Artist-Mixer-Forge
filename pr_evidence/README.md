# PR #4 evidence package

This folder contains compact review evidence and reproducibility support for PR #4.

Key files:

- `xy_preset_matrix_part1.png`: first half of the 1/2/4/10 artist preset matrix.
- `xy_preset_matrix_part2.png`: second half of the 1/2/4/10 artist preset matrix.
- `xy_preset_matrix_summary.md`: readable settings, timings, and file list for the matrix.
- `xy_preset_matrix.metrics.json`: machine-readable settings, timings, and image paths for the matrix.
- `source_Anima.Style.Mixer.json`: reviewer workflow used as the source workflow.
- `compare-1024-32-preset-two-sampler-yuchi.png`: same-model two-KSampler success evidence.
- `compare-1024-32-preset-two-sampler-yuchi.metrics.json`: machine-readable same-model two-KSampler metrics.
- `pr4_reported_issues.metrics.json`: metrics behind the reported-issue checks.
- `mode_preset_matrix.md`: detailed mode and preset reference.

Runtime DBs, logs, PID files, raw ComfyUI output folders, and individual matrix cells are local scratch data and are intentionally ignored.
